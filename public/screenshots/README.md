# Screenshots Directory

Place your app screenshots here to display them on the homepage preview section.

## Required Screenshots

1. **calendar-dashboard.png** - Screenshot of the calendar/dashboard view
2. **reports.png** - Screenshot of the reports/analytics view
3. **budgets.png** - Screenshot of the budgets view
4. **goals.png** - Screenshot of the savings goals view

## How to Take Screenshots

1. Run your app locally
2. Navigate to each view in your dashboard
3. Take screenshots (use browser DevTools or screenshot tool)
4. Save them with the exact filenames listed above
5. The homepage will automatically display them

## Image Recommendations

- Use PNG format for best quality
- Recommended size: 1920x1080 or similar aspect ratio
- Make sure screenshots show realistic data for better presentation
- Consider cropping to focus on the main content area
